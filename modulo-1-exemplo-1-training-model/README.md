# Exemplo 1 — Treino / modelo (TensorFlow.js)

Projeto **Node.js** com **`@tensorflow/tfjs-node`**: base para carregar dados, definir um modelo simples e treinar ou executar inferência em ambiente servidor.

## Como rodar

```bash
npm install
npm start
```

O script `start` usa `node --watch` em `index.js` para recarregar ao editar.

## O que observar

- Fluxo típico: tensores, normalização e uma rede pequena alinhada ao conteúdo teórico da disciplina.
